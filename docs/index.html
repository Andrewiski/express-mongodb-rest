index

Description.

index(options: Object): function
Parameters
options (Object = {}) options for this function.
Name Description
options.mongodb Object (default {}) options for MongoDB database.
options.mongodb.connection string (default process.env.MONGODB_CONNECTION||'mongodb://localhost:27017') Default MongoDB connection string .
options.mongodb.database string (default process.env.MONGODB_DATABASE||'test') Default database name.
options.mongodb.collection string (default process.env.MONGODB_COLLECTION||'express_mongodb_rest') Default collection name
options.mongodb.method string (default process.env.MONGODB_METHOD||'find') Default collection method name
Returns
function: Express middleware compatible with {@link http://expressjs.com/en/api.html#app.use app.use} .
Example
var express = require('express');
var mongoREST = require('express-mongodb-rest');

// (options) Setup mongodb options
var REST = mongoREST({})

// (app) Create express app with middleware
var app = express();
app.use(REST);
app.listen(3000);
